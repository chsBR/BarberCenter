# BarberCenter
_Sistema de Gestão de Barbearia_

## Status do projeto
- Em desenvolvimento

## Tecnologias aplicadas
- Java
- MySQL
- Git
- GitHub

## Time de desenvolvedores
- Projeto desenvolvido individualmente

## Objetivo do software
- O sistema tem como objetivo gerenciar uma barbearia, permitindo o controle de usuários, agendamentos, pagamentos, avaliações e fidelização de clientes

## Funcionalidades do sistema

### Cadastro e login de usuários
- Cadastro de clientes, barbeiros e gerente

### Agendamentos
- Criação de agendamentos com escolha de serviço, barbeiro, data e horário
- Visualização de horários disponíveis
- Cancelamento e conclusão de agendamentos

### Controle por tipo de usuário
- Cliente pode agendar, cancelar e avaliar atendimentos
- Barbeiro pode visualizar e concluir seus agendamentos
- Gerente pode visualizar todos os agendamentos, cancelar e concluí-los

### Pagamentos
- Registro da forma de pagamento ao concluir o atendimento
- Dinheiro, cartão, Pix ou estrelas

### Fidelização
- Cada atendimento gera estrelas para o cliente
- As estrelas podem ser usadas como forma de pagamento
- Cada estrela equivale a 5 reais e deve ser utilizada para pagamento integral do serviço
- Visualização de estrelas na tela de menu

### Avaliações
- Clientes podem avaliar atendimentos concluídos
- Cálculo de média de avaliações por barbeiro

### Notificações
- Aviso ao cliente quando possui agendamento no dia atual
